wp-cli pear channel

Registering the channel: 
pear channel-discover andreascreten.github.com/wp-cli
Listing available packages: 
pear remote-list -c wp-cli
Installing a package: 
pear install wp-cli/package_name
Installing a specific version/stability: 
pear install wp-cli/package_name-1.0.0
pear install wp-cli/package_name-beta
Receiving updates via a feed: 
http://andreascreten.github.com/wp-cli/feed.xml
